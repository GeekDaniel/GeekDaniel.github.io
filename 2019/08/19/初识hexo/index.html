Hexo初次上手

Hexo 目录结构简介

  • _config.yml 是总揽大局的配置文件，配置一下目录themes(主题)。
  • themes 主题的存放目录

主题快速配置

下载开源的主题，可通过快速配置（修改_config.yml）适配到本地。通过以下步骤可以做到。

  1. 下载主题

    hexo主题模块 提供了各种开源主题。每个主题都对应着一个开源项目。直接clone到本地项目根目录下的themes文件夹即可。

  2. 修改_config.yml 的theme 属性。

  3. 重新启动服务hexo server 即可生效。
